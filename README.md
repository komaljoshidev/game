# game
A simple implementation of the classic Rock, Paper, Scissors game built with HTML, CSS, and JavaScript. This game allows you to play against the computer, where the computer randomly selects its choice, and the game shows you whether you won, lost, or tied.
Features:
->Player vs. Computer: Play against an AI opponent that makes random choices.
->Responsive UI: A clean and simple user interface that works across devices.
->Game Outcome: Displays the result of each round (Win, Lose, Draw).
->Multiple Rounds: Play multiple rounds and keep track of the results
